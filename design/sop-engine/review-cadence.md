## Schedule

| Review | Frequency | Covers | Duration | Led By |
|--------|-----------|--------|----------|--------|
| Daily Startup | 06:00 daily | L4 Tasks | 5 min | Agent (auto brief) |
| Daily Close | 20:00 daily | L4 Tasks | 5 min | Agent (auto summary) |
| Weekly | Friday | L3 Projects | 30 min | Agent-assisted, human-led |
| Monthly | Last business day | L2 Objectives | 30 min | Agent-assisted, human-led |
| Quarterly | Last week of quarter | L1 Areas + L2 | 60 min | Human-led, agent prepares |
| Annual | Year-end | L0 Vision | 60 min | Human-led |

## Daily Startup Brief
Top 3 priorities, input queue count, overdue tasks, today's schedule, overnight agent completions, calendar events, blockers.

## Daily Close
Tasks completed/remaining, key result updates, unfinished tasks → auto-reschedule.

## Weekly Review Checklist
1. Process inbox (orphan tasks assigned or dismissed)
2. Project status check (on track / at risk / blocked)
3. Input queue cleanup (stale > 5 days)
4. Dependency check (bottlenecks)
5. Principle capture (decisions → principles)
6. Next week prep (time blocks, agent assignments)
7. Agent review (quality, autonomy adjustments)

## Quarterly Planning (Multi-session)
1. Score outgoing objectives (completed/failed/deferred)
2. Set new objectives + key results
3. Decompose into projects, assign owners
4. Update time blocks, agent assignments

## Configuration
Users can customize: review times, weekly review day, DND hours, review depth (minimal/comprehensive). Stored in settings.
