Merges external messages (email, WhatsApp, Telegram, Discord) and internal input queue.

## Key UX
- Two-column: message list (left) + detail/reply panel (right)
- Filtering: channel, area, urgency, type (message/verify/decide/create)
- Batch actions: approve all, snooze until Monday
- Quick actions: one-click approve, reply, escalate
- Thread view: collapsed with unread indicators
- Agent attribution on input queue items
- Inline preview for deliverables
