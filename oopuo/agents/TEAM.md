## Agent Requirements
Each agent must have:
- Own memory files (self-evolving)
- Clear scope and goals
- Metrics to track
- Ability to spawn sub-tasks
- Report to Kira on heartbeats

---

## Spawn Protocol
1. Define agent purpose and scope
2. Create memory structure
3. Set up heartbeat schedule
4. Initial briefing from Kira
5. First task assignment
